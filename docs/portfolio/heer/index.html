Heer

Built at Ideatolife. Heer is a moment-based rewards SDK which adds value and innovation to the offline shopping experience. It uses an online platform which will engage with the target audience through personalized rewards. The system relies on the estimote beacons installed in the mall shops. The platform provide the following features:

  • Personalized online/offline experience for customers
  • Store analytics
  • Moment based customer engagement
  • Shopper Behavior
